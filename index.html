算法学习笔记（一）

方法

我打算着重看两个地方：

  1. 这个题目里面有哪些知识点？先去把相关内容学习一遍，做过两道例题，再回过头来想这道题。
  2. 标准库里有哪些好用的函数，目标是对它们有个大概的印象，做题的时候可以想不起具体的用法但是要知道有可用的函数。

学习

当你发现大家都摆烂：

这个题还可以 -> （写完后）哦怪不得，看错题了 -> 为什么写不对 -> 好繁琐 -> 我不想写了 -> 我们换个题吧 -> 完了我一道题没搞出来 -> 我是不是智商不太行 -> （看排行榜）-> 吼吼原来大家都差不多 -> 对自己充满信心。

中国古典艺术精神

上课笔记

第六周第二节

对于立人，李老师的核心观点之一是“自由之精神，独立之思想”。就像上课所举的“民”与“人”的例子，从溯源的角度，“民”的甲骨文是由一只眼睛和一支指向眼睛的矛组成的，意思是刺瞎奴隶的眼睛，防止他逃跑，对民的要求总是遵循一定的规章制度，而作为一个人，自己要足够独立，发表的观点需要有理有据，不随波逐流。

核心观点之二是要做一个有温度的人。浩如烟海的历史古籍通常记录的只是王侯将相的英雄事迹，而其他普通百姓的姓名都不会出现在书里。《红楼梦》的伟大之一，就是它写的是普通民众的生活，并且真的做到了写的是千人千面。比起宏大叙事，身边人的生活可能更是我们需要投入精力去关心的，读书的时候我们也要带着温度和感情，去关注书里普通人的生活。
